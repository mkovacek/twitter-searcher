# twitter-searcher

<p>Web app that simulates twitter search</p>
<p>NodeJS, ExpressJS, AngularJS</p>

<h3>Prerequisites:</h3>
<p>Node, bower, grunt</p>

<h3>Instructions:</h3>
  <p>1. git clone https://github.com/mkovacek/twitter-searcher.git</p>
  <p>2. cd twitter-searcher</p>
  <p>3. npm install</p>
  <p>4. bower install</p>
  <p>5. grunt serve</p>
